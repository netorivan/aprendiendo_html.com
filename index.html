¿Que es html 5?

HTML5 es la última evolución de la norma que define HTML. La terminología común "HTML5" se refiere al lenguaje de marcado HTML o a uno de los tres formatos de HTML5. HTML5 es un estándar para la web que define el quinto lenguaje de marcado de hipertexto (HTML) y una aplicación de interfaz de programación de aplicaciones (API) para el navegador web. Es una tecnología clave del World Wide Web Consortium (W3C) y de la Web Hypertext Application Technology Working Group (WHATWG). HTML5 es la quinta versión de HTML y ofrece nuevas características que ayudan a los desarrolladores a crear aplicaciones web más sofisticadas.

¿Que son las etiquetas?

las etiquetas son las que le dan estructura a la pagina web, estas son las que le dicen al navegador como debe mostrar la informacion, por ejemplo, si queremos que un texto se vea en negrita, debemos usar la etiqueta strong, si queremos que un texto se vea en cursiva, debemos usar la etiqueta em, si queremos que un texto se vea en subrayado, debemos usar la etiqueta u, y asi sucesivamente, tambien averiguando en internet pude ver que para una mejor organizacion de la pagina web, se pueden usar las etiquetas como header, footer, section, article, nav, aside, y main, que le dan el significado al navegador web en el que estes y es mas eficas la busqueda y gasta menos rcursos del sistema

  • OJO! las etiquetas deben ser cerradas, es decir, si abrimos una etiqueta, debemos cerrarla
  • las etiquetas son para una estructura semantica osea para ayudar a los navegadores y/o para ayudar a una mejor organizacion en el codigo.
  • no es para organizar la visualizacion del contenido
