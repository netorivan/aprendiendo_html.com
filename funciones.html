los elementos de formato, se diseñaron para mostrar tipos especiales de texto, como veremos a continuacion:

Elemento de formato (b)

este exto tiene como formato el tag (b) y hace referencia a un tono mas oscuro las letras, ahora veamos como se nota sin este tag y como podemos notar en este texto ya no remarca en negrita

Elemento de formato (strong)

este exto tiene como formato el tag (strong) y hace referencia a un tono mas oscuro las letras, ahora veamos como se nota sin este tag y como podemos notar en este texto ya no remarca en negrita

Resumen

En resumen, los elementos de formato como b y strong son útiles para destacar texto, pero tienen diferencias semánticas importantes. strong se usa para remarcar que algun texto es mas importante por ende, le va a indicar al navegador que lo localice primero

pasa los mismo que con lo anterior. los elementos de formato como b y strong son útiles para destacar texto, los mismo con i y em, pero tienen diferencias semánticas importantes. strong se usa para remarcar que algun texto es mas importante los mismo que em por ende, le va a indicar al navegador que lo localice primero y tambien al lector

ESTILOS CSS

Resumen

los estilos css, son una forma de darle estilo a las etiquetas de html, como lo son los elementos de formato, pero con css podemos darle un estilo mas personalizado, como cambiar el color, el tamaño, la fuente, el margen, el padding, entre muchas otras cosas

a partir de este segmento separado por la etiqueta section, empiezamos con los estilos css una introducion basica a los estilos css

las estiquetas como (b,strong,em,mark,del, i, small.) pueden ser modificadas con css, pero son etiquetas de HTML asi como tambien, podemos modificar.
